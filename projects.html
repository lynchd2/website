Danebook

Developed a fully functional social-networking web application using Ruby on Rails with a RESTful approach. The project uses PostgreSQL as a database, and utilizes Amazon S3 to store user uploaded images. Features include commenting and liking on images and posts, uploading photos to a user's profile, commenting on posts and a user's wall, and "friending" other users.
