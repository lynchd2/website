Twitchtv JSON API

A project inspired by FreeCodeCamp. Using the Twitch API, JSON data is gathered and used to determine whether a user is online, offline, or non-existent. A user's icon, the game they are playing and their status are posted if the user appears online.

HTML & CSS Javascript JQuery AJAX

Wikipedia Viewer

Another project inspired by FreeCodeCamp. Using the Wikipedia API, a user inputs a search term and the a list of ten results from Wikipedia, as well as the first couple sentences from the description of each result, are listed. A user can click anywhere in each result to be redirected to the corresponding article on Wikipedia.

HTML & CSS Javascript JQuery AJAX
